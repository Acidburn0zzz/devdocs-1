Request

{
    "type": "related"
}

[
    {
        "code": "position",
        "type": "int"
    }
]

Request and response parameters

Request parameters:
  • type (Change type manually!) TBD.

    Parameters should be specified manually.

Response codes

Normal response codes:
  • SUCCESS_CODE - SUCCESS_DESCRIPTION
